# OS Simulator - Spring 2026

A Python-based Operating System simulator built with Tkinter for PMU.

## Features
- **CPU Scheduling**: FCFS, SJF (Preemptive/Non-Preemptive), and Round Robin.
- **Memory Allocation**: First Fit, Best Fit, and Worst Fit strategies.
- **Page Replacement**: FIFO, LRU, and Optimal algorithms.

## How to Run
1. Ensure you have Python installed.
2. Run the main file:
   ```bash
   python main.py
